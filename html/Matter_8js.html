Plugin

Installs the given plugins on the Matter namespace. This is a short-hand for Plugin.use, see it for more information. Call this function once at the start of your code, with all of the plugins you wish to install as arguments. Avoid calling this function multiple times unless you intend to manually control installation order. use

Parameters
...plugin{Function} The plugin(s) to install on base (multi-argument).
used = []

The plugins that have been installed through Matter.Plugin.install. Read only.

uses = []
